## DMARC export (Outlook -> CSV)

Short description:
- Reads unread emails from a selected Outlook folder.
- Saves and extracts DMARC attachments.
- Parses XML reports and generates two CSV files (`filereport.csv` and `rowreport.csv`).

Detailed documentation:
- [dmarc_export/_script_dmarc_outlook_to_csv.md](dmarc_export/_script_dmarc_outlook_to_csv.md)

## Exportacion DMARC (Outlook -> CSV)

Descripcion corta:
- Lee correos no leidos desde una carpeta de Outlook seleccionada.
- Guarda y extrae adjuntos DMARC.
- Procesa reportes XML y genera dos archivos CSV (`filereport.csv` y `rowreport.csv`).

Documentacion detallada:
- [dmarc_export/_script_dmarc_outlook_to_csv.md](dmarc_export/_script_dmarc_outlook_to_csv.md)
